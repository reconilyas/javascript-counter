# JavaScript Counter

A simple counter project built using HTML, CSS, and JavaScript.

I created this project to practice working with the DOM, handling button events, and updating elements dynamically using JavaScript.

## Features

- Increase the counter value
- Decrease the counter value
- Reset the counter
- Change the counter color depending on the value

## Technologies Used

- HTML
- CSS
- JavaScript

## What I Learned

- Selecting HTML elements with JavaScript
- Using event handlers
- Updating page content dynamically
- Working with variables and conditions
